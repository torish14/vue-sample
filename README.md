# Vue.js のサンプルプロジェクトです!

## computed
算出プロパティ - 関数によって算出されたデータ

## created
ライフサイクルフック - Vue.js の「おはよう」から「おやすみ」までの一定のサイクルを表すもの
フックとは、決まったタイミングに処理を割り込ませること
